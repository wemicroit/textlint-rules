# textlint-rule-cspell



## Install

Install with [npm](https://www.npmjs.com/):

```shell
npm install textlint-rule-cspell
```

## Usage

Via `.textlintrc.json`(Recommended)

```json
{
    "rules": {
        "cspell": true
    }
}
```

Via CLI

```
textlint --rule cspell README.md
```
